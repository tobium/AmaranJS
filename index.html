Installation

Minimal Setup

Go to dist directory and copy all the folders to a new destination, which will be your project home.

It’s not required, but i recommend placing CSS files in <head> and JavaScript files and initialization code in the footer of your site (before the closing </body> tag).

<link rel="stylesheet" href="/css/amaran.min.css">

jQuery required for Amaran JS.If you already have jquery.js on your site, don’t include it a second time, or use jQuery.noConflict(); mode.

You can add jQuery via Google CDN

 <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

Place Amaran JS after jQuery

<script src="/js/jquery.amaran.min.js"></script>

So minimalistic setup look like this

<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
    <link rel="stylesheet" href="amaran/css/jquery.amaran.min.css">
</head>
<body>
    <p>My Content</p>
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script src="amaran/js/jquery.amaran.js"></script>
</body>
</html>

Basic Usage

We are ready to create our first example.Very very simple usage of Amaran JS.

First of all we must add our code inside jQuery document ready function.$(function(){ });

message options available for all themes and you can use html code for message option.

$(function(){
    $.amaran({content:{'message':'My first example!'}});
});

If you add this code inside $(function(){ }) function it will automatically popsup when page loads.

This code using default theme.If you want to use default theme you dont have to specify theme option.

Lets create a notification with awesome theme.

$.amaran({
    content:{
        title:'My first funcy example!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok'
});

Usage

$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok'
});

You can specify a theme by passing theme : '' option to Amaran JS object.

For "awesome" theme you can call theme and variation with theme 'awesome <VARIATION NAME>' e.g : theme:'awesome ok' ortheme:'awesome error'.We have 3 content options title,message and info.title represent first line,message second line and info reperesent third line.icon options you can specify icon."awesome" theme using Font Awesome icons.Icons list can be found in this http://fontawesome.io/icons/ adress.

e.g For envelope icon just pass icons:'fa fa-envelope'

Creating your own variations

.amaran.awesome.<VARIATION NAME> p.bold{
    color:<COLOR CODE>;
}
.amaran.awesome.<VARIATION NAME> .icon {
    background-color:<COLOR CODE>;
    color:#fff;
}

Just change <VARIATION NAME> with your variation name and <COLOR CODE> with your color code and add in your main css file.

e.q

.amaran.awesome.pink p.bold{
    color:#D704DE;
}
.amaran.awesome.pink .icon {
    background-color:#D704DE;
    color:#fff;
}

Options Overview

In Effect

Default : inEffect:fadeIn

Options :
inEffect:fadeIn
inEffect:slideLeft
inEffect:slideRight
inEffect:slideTop
inEffect:slideBottom

Notifications appear effect

    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok',
    inEffect:'slideRight'
});

Out Effect

Default : inEffect:fadeOut

Options :
outEffect:fadeOut
outEffect:slideLeft
outEffect:slideRight
outEffect:slideTop
outEffect:slideBottom

Notifications appear effect

$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok',
    outEffect:'slideRight'
});

delay

Default : delay:3000

Default auto hide time is 3 seconds you can specify auto hide time with delay option.

delay : <milisecond>

Example
$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok',
    delay:10000
});

Creating Themes

First of all decide your html structure for your template.


    USER INFO
    

        Hakan
        ERSU
        My Message

Now lets use themeTemplate option.All we have to do pass our html structure with placeholders we choose.And pass our content object with values.

But first you have to pass your theme name on your content object.themeName:

Its necessary for better styling.

$.amaran({
        content:{
            themeName:'myTheme',
            firstname:'Hakan',
            lastname:'ERSU',
            message:'Hi'
           },
        themeTemplate:function(data){
          return '
USER INFO
'+data.firstname+''+data.lastname+''+data.message+'
';
        },
        position:'top right'
    });


Lets style our notification.

.amaran.myTheme{
  background: #ced4dc;
  font-size:13px;
  padding:0px;
  margin: 5px 10px;
}

  padding:2px 4px 2px 10px;
}
.amaran.myTheme span{
  display: block;
}
.amaran.myTheme .inner-content{
  padding:10px;
  color:#222;
}

Using With Animate CSS

Animate.css is a cross-browser library of CSS animations.

Animate.css already included in AmaranJS css folder but you can still download current version from offical page.

Just play with it tons of cool animation you can create.

        bgcolor:'#8e44ad',

        message:'Lets see with animate.css'
       },
    theme:'colorful',
    position:'bottom right',
    closeButton:true,
    cssanimationIn: 'swing',
    cssanimationOut: 'bounceOut'

});
